# InstagramClient

This is an Android demo application for displaying most popular images using [Instagram API]
(http://instagram.com/developer/endpoints/media/#get_media_popular)

Time spent: 6 hours spent in total

Completed user stories:

Required: User can scroll through current popular photos from Instagram
Required: For each photo displayed, user can see details like Graphic, Caption, Username,timestamp
Advanced: Relative timestamp, like count, user profile image,comments count
Advanced: Display each photo with the same style and proportions as the real Instagram
Advanced: Display each user profile image using a RoundedImageView
Advanced: Add pull-to-refresh for popular stream using SwipeRefreshLayout
Bonus:    User can view all comments for an image within a separate activity
